# GitHub Pages Setup Guide for Hair Accessories Demo

## Step-by-Step Instructions

### **Step 1: Create GitHub Account**
1. Go to [github.com](https://github.com)
2. Click "Sign up" and create your free account
3. Verify your email address

### **Step 2: Create New Repository**
1. Click the **"+"** icon in top right corner
2. Select **"New repository"**
3. Fill in repository details:
   - **Repository name**: `hair-accessories-demo`
   - **Description**: `Demo e-commerce website for hair accessories business`
   - **Visibility**: Choose **Public** (required for free GitHub Pages)
   - **DO NOT** check "Add a README file" (we already have files)
4. Click **"Create repository"**

### **Step 3: Upload Demo Files**
1. In your new repository, click **"uploading an existing file"**
2. **Drag and drop** all files from your `demo` folder:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `README.md`
   - `share-instructions.md`
   - `github-setup-guide.md`
3. **Commit changes**:
   - Fill in "Add changes" with something like "Initial demo upload"
   - Click **"Commit changes"**

### **Step 4: Enable GitHub Pages**
1. In your repository, go to **Settings** (tab)
2. Scroll down to **"Pages"** in the left menu
3. Under "Build and deployment":
   - **Source**: Select **"Deploy from a branch"**
   - **Branch**: Select **"main"**
   - **Folder**: Select **"/ (root)"**
4. Click **"Save"**

### **Step 5: Get Your Live URL**
1. Wait 1-2 minutes for deployment
2. Go back to Settings > Pages
3. Your live URL will appear: `https://[your-username].github.io/hair-accessories-demo/`

### **Step 6: Share with Your Friend**
Send this URL to your friend:
```
https://[your-username].github.io/hair-accessories-demo/
```

## Quick Alternative: GitHub Desktop (Easier)

### **If You Prefer Desktop App:**
1. Download [GitHub Desktop](https://desktop.github.com/)
2. Install and sign in
3. **File > Clone Repository** (choose "Create new repository")
4. Browse to your `demo` folder
5. **Repository name**: `hair-accessories-demo`
6. Click **"Create Repository"**
7. Click **"Publish Repository"**
8. Choose **Public** and check **"Keep this code private"**
9. In GitHub Desktop, click **"Repository > Repository Settings"**
10. Go to **Pages** section and enable as above

## Troubleshooting

### **Common Issues:**

#### **"Page not found" error:**
- Wait 5-10 minutes after enabling Pages
- Check that repository is **Public**
- Verify you selected **main branch** and **root folder**

#### **Images not loading:**
- Make sure all image URLs use **https://**
- Check file names match exactly (case-sensitive)

#### **Styles not applying:**
- Ensure `styles.css` is in the root folder
- Check the CSS link in `index.html`

## Advanced Options

### **Custom Domain (Optional):**
If you want a professional-looking URL:
1. Buy a domain (e.g., `bellahair.rs`)
2. In repository Settings > Pages > Custom domain
3. Add your domain name
4. Update DNS settings as instructed

### **Password Protection (Advanced):**
GitHub Pages doesn't support password protection, but you can:
- Use a private repository and add collaborator access
- Add a simple password prompt in JavaScript
