# Bella Hair Accessories - Demo Website

## Overview
This is a demonstration e-commerce website for a Serbian hair accessories business. It showcases modern web development practices and e-commerce features that can be implemented for your friend's business.

## Features Demonstrated

### 🛍️ **E-commerce Functionality**
- Product catalog with categories (Scrunchies, Hair Clips, Headbands)
- Shopping cart with add/remove items
- Product quick view modal
- Category filtering
- Price display in Serbian Dinars (RSD)

### 🎨 **Design Elements**
- Modern, responsive design using Bootstrap 5
- Custom pink color scheme suitable for hair accessories
- Smooth animations and hover effects
- Mobile-first responsive design
- Professional product cards with image overlays

### 📍 **Serbian Market Adaptation**
- Prices in RSD (Serbian Dinars)
- Belgrade-based business messaging
- Local delivery information
- Serbian market considerations

### 📱 **User Experience**
- Smooth scrolling navigation
- Interactive product filtering
- Shopping cart notifications
- Contact form
- Social media integration placeholders

## Technical Implementation

### **Frontend Technologies**
- **HTML5**: Semantic markup

### **Key Features**

## Customization Options

### **Brand Customization**
```css

:root {
    --pink-color: #FF69B4;  /* Change to brand color */
    --pink-light: #FFB6C1;  /* Light variant */
    --pink-dark: #FF1493;   /* Dark variant */
}
```

## Platform Comparison

### **Shopify Implementation**
- Similar design can be achieved with Shopify themes
- Built-in payment processing
- Inventory management included
- Monthly cost: $29-299

### **WordPress/WooCommerce**
- More customization freedom
- One-time hosting costs
- Requires more technical management
- Estimated cost: $50-100/month

### **Custom Development**
- Complete control over features
- Higher initial development cost
- Ongoing maintenance required
- Estimated cost: $1000-5000+
